Hello guys..
There are some steps you should follow to run this project.

1. Click the browse button to select image from "Successful images" folder.
The image will appear on the blank space in between.

2. Before running the 'slider' you should either press 'structural layout' 
or 'logical layout' button,so that image is cropped and saved in a 'Output_folder'.

3. To watch labeled image press 'logical layout' button.

4. If you think there is some noise in image, click 'Remove noise'.
